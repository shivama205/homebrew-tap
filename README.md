# shivama205/homebrew-tap

Homebrew tap for tools I maintain.

## Install

```bash
brew tap shivama205/tap
```

## Formulas

### excalidraw-render

Clean, browser-free renderer for `.excalidraw` files (PNG/SVG).

```bash
brew install shivama205/tap/excalidraw-render
excalidraw-render path/to/diagram.excalidraw
```

Project: https://github.com/shivama205/excalidraw-render
PyPI: https://pypi.org/project/excalidraw-render/
